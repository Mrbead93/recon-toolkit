# Recon Toolkit

A personal recon toolkit built in Python, running on Termux (Android).

## Tools

### Port Scanner
Threaded TCP port scanner with service detection, colour output and auto-saved reports.

### OSINT Username Tool
Multi-platform username recon tool covering 12 platforms including GitHub, GitLab,
Reddit, Twitter, Instagram, TikTok, LinkedIn, npm, PyPI and more.

## Built With
- Python 3
- socket, threading, requests
- Termux on Android

## Status
Active development — part of a structured learning roadmap in Python,
cybersecurity and networking.
